Le projet est sur git https://github.com/matnoel/PythonEF.git

il faut installer sur python :
	numpy
	scipy
	matplotlib
	gmsh
	pandas
	pypardiso (optional)
	numba
    colorama
	scikit-umfpack  (optional)
	scikits-sparse  (optional)


Il faut impérativement ajouter le dossier dans lequel est contenu les modules dans le path
Pour ça on ajoute dans le PYTHONPATH:
	
	sur macOs :
		dans le .zprofile ajouter 
		export PYTHONPATH:"$PYTHONPATH: .../PythonEF/modules"

	sur linux :
		export PYTHONPATH=$PYTHONPATH: .../PythonEF/modules
		
	sur windows : 
		modifier les variables du système -> variables d'environnement -> variables utilisateur
		PYTHONPATH ...\PythonEF\modules

Build gmsh on macos:

	il faut dabord  installer opencascade : sudo port install opencascade
	il faut ensuite installer fltk : sudo port install fltk

	Sur Mac il faut construire le projet gmsh
	https://gitlab.onelab.info/gmsh/gmsh
	Dans le dossier applications:
	git clone 
	cd gmsh
	mkdir build
	mkdir bin
	cd build
	cmake -DCMAKE_INSTALL_PREFIX=/Applications/gmsh/bin -DENABLE_BUILD_DYNAMIC=1 .. 
	make 
	make install

	il faut ensuite ajouter le path vers gmsh.py
	export PYTHONPATH=".../PythonEF/modules:/Applications/gmsh/bin/lib"
